What is your PASS-word?

January 30, 2020 2-minute read
linux • technical • OSS
encryption • gpg • software • terminal • pass • diceware

Ever been frustated remembering so many password for so many services you signed up for, and sometimes only for one time use? If yes, this blog post will change your situation, as it did for me.

Before

Single sign on using third party services leading that entity total control about and knowledge about my presence everywhere.

Using same and weak passwords everywhere, with minor difference which adds to the threat analysis. as even if gets compromised, there are chances, other passwords can be derived from them.

After

Importance of password managers?

Better password management, in this digital era, where people forgets their password like chapter they studied for the exam at viva. It makes it easy for everyone to remember them using password managers. and help them to use as long and random password as they want, and can use different password everywhere.

Setup one for yourself

Options? There are many, have a read on this article. But pass is by far the best terminal based password manager.

Thanks to sayan and kushal for sharing about this at Pycon Hyderabad 2018. I switched to “pass”, a terminal based gpg password manager in Fedora to keep track of all your password in encrypted format using GnuPG.

Now I have stopped using my own passwords for a long time. I use diceware (to know more about ) to generate random phrase and it is proven that strong passwords do not need to have special symbols but should be long enough and random enough to be guessed and bruteforced by anyone.
